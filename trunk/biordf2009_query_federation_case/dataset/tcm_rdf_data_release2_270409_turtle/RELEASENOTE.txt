The original data are published by the TCMGeneDIT team (http://tcm.lifescience.ntu.edu.tw/), available as tab delimiter files. They are distributed under the terms of the Creative Commons Attribution License, given proper citation to the original work published in BMC Complementary and Alternative Medicine [doi:10.1186/1472-6882-8-58]. 

The conversion of this dataset into RDF was supported by Jun Zhao. Please contact Jun for support or any mistakes in the RDF dataset.

Documentation and example queries to this dataset can be found at: http://code.google.com/p/junsbriefcase/wiki/RDFTCMData.

New feature since R1:
* Updated URIs in the dataset with a new domain name: http://purl.org/net/tcm/tcm.lifescience.ntu.edu.tw
* Updated the dataset with the statistic confidence value of the association between entities
* Released the mapping of TCM gene names and their linking to Entrez gene ids


Future development:
* Generate a mapping of the medicine and ingredient names to the DBPedia resource URIs
* Generate a mapping of the gene names used in this dataset to those used by another genomic public data resource
* Update the data content if a major update to the source database would be available
